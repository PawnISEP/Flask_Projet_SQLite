Politique de confidentialité

Votre confidentialité est importante pour nous. Cette page explique comment nous collectons, utilisons et protégeons vos données personnelles lorsque vous utilisez notre plateforme.

1. Collecte des données

Nous collectons les informations suivantes pour vous offrir une expérience personnalisée :

  • Données personnelles : nom, adresse e-mail, numéro de téléphone, etc.
  • Données d'utilisation : historique de navigation, pages visitées.
  • Informations techniques : adresse IP, type de navigateur.

2. Utilisation des données

Les données collectées sont utilisées pour :

  • Fournir les services demandés.
  • Améliorer notre plateforme.
  • Personnaliser votre expérience utilisateur.
  • Envoyer des notifications pertinentes.

3. Sécurité des données

Nous mettons en œuvre des mesures de sécurité rigoureuses pour protéger vos données :

  • Cryptage des données sensibles.
  • Stockage sécurisé sur des serveurs protégés.
  • Accès limité aux employés autorisés uniquement.

4. Vos droits

Vous avez le contrôle total sur vos informations personnelles :

  • Accéder à vos données en nous contactant.
  • Modifier ou mettre à jour vos informations.
  • Demander la suppression de vos données.
  • Refuser certaines utilisations de vos informations.
